A satirical presentation website featuring a "lying AI" that provides hilariously incorrect educational facts with unwavering confidence.

## About

SchoolAI is a humorous demonstration of AI confidence vs. accuracy. This interactive presentation website showcases how AI can sound convincing while being completely wrong.
